Week Ten Notes

Thoughts

I became very intrigued by the idea of custom validation this week in the reading, so I went down the rabbit hole and decided to create a Mad Libs style page that would be able to check the inputs for specific criteria. Initially I wanted to be able to validate parts of speech, but without server based validation processes that looked to be impossible. I found an API that could contain parts of speech information, but it was too complex to develop a function that could reliably pull that information for any term that was entered. So I had to scale back and create an error checking function that simply looked for length on certain words and checked others against numerical ranges or RegExp formats. It was also a good challenge placing and clearing the error messages as appropriate. Please enjoy my Silly Story.
