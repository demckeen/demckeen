A concept I thought would be potentially really useful to me in the future was the section about using a controller to watch a form to append a list. I can see how this pattern could be used for many things, outside of lists even. Of course, this is an example of MVC - which I like as a subject in this chapter that seems to be in some cases about working outside of simple js and incorporating libraries and frameworks. While this MVC pattern is often used in those contexts, it's also something that can be applied directly.

'use strict' const form = document.forms[0]; class Item { constructor(name) { this.name = name; } }

const controller = { watch(form) { form.addEventListener('submit', (event) => { event.preventDefault(); // prevent the form from being submitted this.add(form.name.value); }, false); }, add(name) { const item = new Item(name); view.render(item); } };

const view = { render(item) { const list = document.getElementById('list'); const li = document.createElement('li'); li.innerHTML = item.name; list.appendChild(li); // reset the input field form.name.value = ''; } };

controller.watch(form);
